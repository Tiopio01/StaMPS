# Code of Conduct

This project adopts the Contributor Covenant v2.1:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

All interactions in issues, pull requests, discussions, and related forums.

## Enforcement

Report violations to the maintainers listed in MAINTAINERS.md.
Maintainers act within 7 days.

# Maintainers

The `windows-port/*` branches and `pyccino/StaMPS` fork are maintained by
the PHASE team. Upstream merges back to `dbekaert/StaMPS` are opened by
the same team; acceptance timeline is unpredictable.

## Primary
- pyccino  (fork owner)

## Scope
- Windows build + CI infrastructure
- Python port of csh entry points
- MATLAB compatibility helpers

See `docs/superpowers/specs/2026-04-20-stamps-windows-port-design.md` §3 for
fork-first strategy rationale.
